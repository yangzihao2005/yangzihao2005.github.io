学习任何事情都要先学结构再学原理。GPIO也不例外。

STM32F103ZET6有6组GPIO引脚、112个普通GPIO引脚。我们只需要学习112中的一个即可,其他的基本一致。

## 1.GPIO是什么?
GPIO也叫通用输入/输出。即一个MCU内部,用来做输入输出的控制器电路。
输入输出的本质是输入/输出高低电平,从而获取外设信息,控制外设的目的。通用的意思是给大家用的。其他片上外设控制器(I2C、串口等),直接使用GPIO硬件电路来进行输入输出的方式,叫做复用GPIO。

## 2.GPIO引脚如何板级外设产生关联?
GPIO引脚有2种工作方式,输入/输出。具体看一下GPIO引脚如何与外围器件产生关联。

i.输入:比如外接"按钮",MCU主要是通过感知外部世界的状态变化,比如:按键是否被按下?某个信号是高电平还是低电平?来获取外部信息。
![](/images/img-58.png)
![](/images/img-59.png)

ii.输出:比如外接发光二极管(LED)。MCU控制外部设备(比如:点亮一个LED灯?让蜂鸣器响?控制继电器通断?)
![](/images/img-60.png)
![](/images/img-61.png)
![](/images/img-62.png)

再次,再一次的认识一下GPIO引脚如何与外围器件产生关联?
GPIO引脚会接各种板级外设,通过MCU内部的输入/输出寄存器来进行读/写对应的寄存器,从而达到对引脚输入高低电平/读取引脚高低电平。------核心就是MCU上的寄存器。

## 4.GPIO PIN基本电路结构中的端口保护
明白了IO基本电路结构宏观部分后,来看一下其基本结构中的一部分。I/O保护电路,其实就是双二极管钳位电路。
![](/images/img-68.png)

二极管是一种单向导电的电子元件,只允许电流从正极流向负极,反向则阻断.(电压差形成电流,电流驱动二极管导通。)
![](/images/img-69.png)

钳形保护二极管就是为了防止不合理电压输入道针脚,进而形成大电流,导致击穿MCU内部元器件,本质是一种保护电路!
![](/images/img-70.png)

```
用手触摸MCU引脚时,可能会因为静电,导致针脚电压过大,如果没有保护电路,MCU就寄了!
```
